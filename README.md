## stachu

A stack-like task recorder

~~~bash
$ ./stachu
stachu > fufufu
1. fufufu
stachu > bababa
1. bababa
2. fufufu
stachu > rarara
1. rarara
2. bababa
3. fufufu
stachu > rm 2
1. rarara
2. fufufu
stachu > rm 1
1. fufufu
stachu > rm 1
stachu >
stachu >
~~~

#### Install and execute
~~~bash
sh dev.sh
./stachu
~~~
